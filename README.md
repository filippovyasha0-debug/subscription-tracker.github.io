SubEnd — трекер подписок

**SubEnd** — это простое и удобное веб-приложение для контроля ежемесячных и годовых подписок. Оно помогает не забывать о списаниях, видеть реальные расходы и вести историю платежей.  
Проект создан как студенческая работа и распространяется бесплатно.

Возможности

- ➕ **Добавление подписок** — название, стоимость, дата следующего списания, периодичность (месяц / год).
- ✏️ **Редактирование и удаление** — любую подписку можно изменить или удалить.
- 📊 **Статистика расходов** — автоматический подсчёт трат в месяц и в год.
- ⏰ **Дни до списания** — цветовая индикация: сегодня, завтра, скоро (≤3 дней), просрочено.
- 📜 **История платежей** — запись каждого оплаченного списания с возможностью очистки.
- 🌙 **Тёмная / светлая тема** — переключение одним нажатием, сохраняется в localStorage.
- 📱 **Адаптивный дизайн** — корректно работает на любых устройствах (ПК, планшет, смартфон).
- 💾 **Все данные в браузере** — не требуется регистрация, сервер или интернет после загрузки.

Технологии

- HTML5, CSS3 (CSS переменные, Grid, Flexbox)
- JavaScript (ES6)
- LocalStorage для хранения подписок и истории
- Адаптивная вёрстка
